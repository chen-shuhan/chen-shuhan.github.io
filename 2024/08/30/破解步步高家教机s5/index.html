破解步步高家教机s5

发表于 2024-08-30
步步高家教机S5的降级、获取root权限及隐藏Root的详细步骤如下：首先，通过官网下载并降级到1.4.3版本；然后，利用USB调试功能连接电脑进行一系列操作，包括刷入TWRP和Magisk以获取root权限。接下来，通过安装《安装狮》和《OPPO软件商店》解除应用安装限制，并使用《EEBBK TOOLBOX》冻结家长控制相关组件。最后，在Magisk中设置排除列表来隐藏Root，避免部分应用因检测到Root而无法正常使用。整个过程需要谨慎操作，确保了解每个步骤的具体作用和潜在风险。

引入主题


这是一台步步高s5家教机。作为一台家教机，它可以下载教辅、查看网课，甚至可以登录微信和QQ。
但它只能是一台家教机吗？

这个配置已经能够运行很多游戏了，加上这硕大的屏幕，这简直就是我的音游神机啊！
但是家教机本身有安装限制和家长控制，总之就是不能无忧无虑地玩耍。所以就有了这篇文章。

将家教机降级到1.4.3版本

  1. 网络搜索步步高，进入步步高官网
  2. 点击右上角“下载”进入机型选择界面，选择s5
  3. 在模块一栏选择“软件升级及工具”，选择家教机_S5_V1.4.3_Recovery恢复工具，点击下载（需要较长时间）
  4. 解压下载的rar文件，根据其中doc教程的指示，将文件复制到家教机并进行降级

打开USB调试

  1. 打开设置，在左栏选择应用程序，将右侧字母表拖动到x，点击相机
  2. 点击权限，禁用相机的任一权限（不顾警告）
  3. 打开相机，弹出授权窗口。点击禁止，点击去设置
  4. 点击蓝牙，再点击WIFI，点击右上角出现的放大镜
  5. 搜索开发者选项
  6. 下滑，找到USB调试，点击打开

电脑软件准备

  • 下载并安装《安卓手表ADB实用工具箱》（点击下载），为后续ADB调试设备做准备
  • 下载并解压《步步高家教机S5-MIUI12.5体验包V1》（点击下载）,为edl刷入TWRP做准备
  • 下载并安装《高通9008驱动》（点击下载），为edl刷入TWRP做准备

edl模式下刷入TWRP

魔改《步步高家教机S5-MIUI12.5体验包V1》中的“点我刷入.bat”

这个包可以帮我们刷入整个MIUI系统，但是在这里我只是借助它刷入TWRP，所以需要魔改。刷入MIUI后有声音撕裂、亮度过低等BUG，所以不推荐刷入。

  1. 使用记事本打开点我刷入.bat
  2. ctrl+f打开搜索，搜索:FLASH,删除该部分ECHO.重启Recovery...之后的代码，如图所示
  3. ctrl+s保存文件

adb重启设备至edl模式

  1. 使用数据线连接电脑与家教机
  2. 电脑打开《安卓手表ADB实用工具箱》，此时家教机弹出授权窗口，把勾打上，点击允许
  3. 电脑显示出设备后，点击其他功能，点击ADB命令行
  4. 在命令行输入adb reboot edl让家教机进入edl刷机模式

使用《步步高家教机S5-MIUI12.5体验包V1》刷入TWRP

  1. 打开工具包，将修改后的“点我刷入.bat”拖入刚才打开的命令行。如果电脑弹出授权窗口，点击允许
  2. 进入刷入工具的菜单后，选择2，刷入TWRP

使用TWRP刷入magisk获取root

  1. 下载magisk。这个应用可以更改系统的任何文件，所以务必从Github或其他可信渠道下载
  2. 更改magisk安装包的扩展名为.zip
  3. 家教机长按音量+和电源键，直到进入TWRP,滑动底部滑块进入主界面
  4. 电脑复制刚才的magisk文件到内部存储根目录
  5. 点击安装，选择magisk文件，滑动滑块安装
  6. 刷入成功后，点击重启系统
  7. 查看桌面是否有magisk
  8. 点击打开

安装《安装狮》和《OPPO软件商店》解除安装限制

  1. 下载安装狮
  2. 电脑打开《安卓手表ADB实用工具箱》，将下载的《安装狮》安装程序拖动到“安装应用”按钮上，点击安装
  3. 以相同步骤安装《OPPO软件商店》
  4. 安装完成后，家教机打开《安装狮》，此时弹出授权窗口，点击允许
  5. 打开《OPPO软件商店》，下载任一应用
  6. 下载完成后，会让你选择打开方式，选择“安装狮-Root”，把勾打上

解除家长远程控制

  1. 下载《EEBBK TOOLBOX》（点击下载）
  2. 打开《EEBBK Toolbox》，点击功能，点击应用程序管理，点击冻结步步高服务组件，点击确定。如果出现授权窗口，点击允许。

隐藏Root

因为在root环境下，一些应用无法正常使用（如《国家反诈中心》《酷狗音乐》），因此需要对这些应用隐藏Root

  1. 打开magisk，点击右上角设置
  2. 打开Zygisk和遵循排除列表
  3. 点击配置排除列表
  4. 点击应用图标，再点击右侧的勾，使应用上方的长条为满
    如图长条未满，请重新操作
    如图长条已满，请继续操作
  5. 重复4步骤，直到除《安装狮》《EEBBK Toolbox》外的应用被全部勾选
